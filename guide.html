OPERATING MANUAL
1. The Dashboard (Invoices)

The dashboard is your home screen. It lists all active Quotes and Invoices sorted by date.

Searching & Filtering

Use the top search bar to find documents by Client Name or Ref Number (e.g., "RB-2025").

Action Buttons

  • Edit : Opens the Editor to make changes.
  • PDF : Instantly downloads the formal PDF contract.
  • Copy : Duplicates an invoice (useful for creating "Option B" quotes).
  • Delete : Permanently removes the document.
2. Creating a Quotation

This is where you build the contract. Data saves automatically every time you stop typing.

Step A: Project Details

  • Select Client: Type a name manually OR click the icon to pick from your saved Customer list.
  • Doc Type: Switch between "Quotation" (Draft) and "Invoice" (Final Bill).

Step B: Adding Work Items

You build the quote by adding Sections (e.g., "Kitchen", "Master Bed") and then adding items inside them.

Pro Tip: Click the "Library" button inside a section to instantly insert standard items (like "Supply & Lay Tiles") with their pre-set prices.

If you need a custom item, click "Blank Item" and type the description manually.

Step C: VO (Variation Orders)

Crucial: If a client changes the scope AFTER signing, do not delete the original items.
  1. Scroll to the bottom "Payment & Revisions" box.
  2. Type a note (e.g., "Added electrical points").
  3. Click Create VO Snapshot. This saves a hidden copy of the current state before you make changes, protecting your original contract history.
3. Using on Mobile Phone

The system changes its layout when used on a phone to act like an app.

MOBILE MODE

Key Differences

  • Navigation: Use the Menu Button at the top right to switch between Invoices, Customers, and Library.
  • Tables: Invoices appear as "Cards" instead of rows. Tap a card to see actions.
  • Editor: The "Qty/Unit/Price" inputs stack vertically to make them easier to tap with your thumb.
4. Library & Customers

The Library

This saves you from typing the same thing twice. Go to the Library tab to manage standard rates.

  • Manage Mode: Inside the editor, opening the library allows you to toggle "Manage Mode" to add/delete items on the fly without leaving the invoice.

Customers

You can upload an Excel file of customers to bulk-import them.

Tracking Balance: Click the icon next to a customer to see every invoice they have ever received and their total outstanding payment.
5. Owner Features

These features are hidden or restricted.

Profit Calculation

Open any invoice and scroll to the bottom black footer bar.

  • Locate the small "Internal Cost" box next to the Grand Total.
  • Enter your actual cost for the job.
  • The system will display your Net Profit immediately below it. (This is never shown on the PDF).

Default Terms

Go to the Settings tab to change the default "Terms & Conditions" text that appears on every new invoice.
