Renobros Interior

System Operating Protocol v2.0
1. Getting Started

This system is a cloud-based web application. It runs in any modern browser (Chrome/Safari) on desktops, tablets, and mobile phones.

Access: Simply open the index.html file. The system connects to the central database automatically.

Logging In

  • Open the application.
  • Enter your company email and password.
  • If you are a new staff member, toggle to "Create Account" to register your ID first.
2. Creating Quotations & Invoices

This is the primary function of the app. All documents save automatically as you type.

Step A: Setup

  • Click New Invoice on the dashboard.
  • Doc Type: Select "Quotation" for new leads or "Invoice" for billing.
  • Client: Click the icon to pick an existing customer, or type a new name manually.

Step B: Building the Scope

  • Add Sections: Click "Add New Section" to create categories (e.g., "Kitchen", "Master Bedroom").
  • Add Items: Inside a section, you have two choices:
    Option 1: Click Library to insert pre-set items (Fastest).
    Option 2: Click Blank Item to type a custom description.
  • Pricing: Enter Qty, Unit, and Price. The totals calculate automatically.

Step C: Revisions (VOs)

If a client changes their mind after the contract is signed, do not overwrite the original.
  • Scroll to the "Payment & Revisions" card.
  • Type a note (e.g., "Added electrical points").
  • Click Create VO Snapshot. This saves a copy of the current state before you make changes.
3. Mobile Usage Guide

The system transforms when opened on a phone to look like a native app.

Key Differences on Phone

  • Navigation: Use the "Hamburger" menu at the top right to switch views.
  • Tables: Invoices and Customers appear as "Cards" instead of spreadsheet rows.
  • Editing: Item rows stack vertically. The "Total Price" for an item appears at the bottom right of the card.
Tip: Tap the "Actions" area on an invoice card to quickly "Quick Download PDF" via WhatsApp or Email.
4. Data Management

Library Management

The library stores your standard rates to speed up quoting.

  • Go to the Library tab.
  • Click New Item to add a standard service.
  • Bulk Edit: Use the "Manage Mode" inside the invoice editor to quickly delete old items or add categories on the fly.

Customer Database

Customers are saved automatically when you import them via Excel or create them manually.

  • Excel Import: In the Customers tab, upload an Excel file with columns: Name, Phone, Email, Address.
  • Tracking: Click the icon on any customer to see their total outstanding balance across all jobs.
5. Admin Protocols

Profit Tracking (Owner Only)

This feature is only visible to the primary admin account.

  • Open an Invoice/Quotation.
  • Scroll to the bottom footer bar.
  • Enter your Internal Cost in the small box next to the Grand Total.
  • The system will calculate the Net Profit dynamically.

PDF Generation

The PDF generator creates a formal contract.

  • It automatically includes the "Terms & Conditions" set in the Settings tab.
  • It generates a signature page automatically.
